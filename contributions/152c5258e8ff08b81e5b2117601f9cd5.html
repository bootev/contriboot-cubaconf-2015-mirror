Mapping Party con OpenStreetMap

Santiago Higuera - @santiagohiguera

Taller y mapping party con OpenStreetMap. OpenStreetMap (OSM) es un proyecto colaborativo para crear un mapa mundi libre y editable. El mapa de OpenStreetMap está hecho a base de las contribuciones de cientos de miles de voluntarios que incorporan información y la ponen a disposición de la comunidad. Toda la información se almacena en bases de datos y se puede visualizar a través de mapas web o mediante consultas y descargas directas desde la base datos. Toda la información se puede consultar, editar o utilizar libremente. Una mapping party, o partida de mapeo, es una actividad de grupo en la que se añaden al mapa de OSM elementos de una determinada zona. En el taller y mapping party aprenderíamos las principales características de OpenStreetMap así como las principales técnicas de mapeo, edición y utilización de los datos de OSM. Puedes acceder al mapa web en http://openstreetmap.org.
